Cancel Order" plugin+ After the installation is complete click "Activate" link (Plugins > Installed Plugins).+ Make desired settings under woocommerce settings >> Wc Cancel Tab.+ You are done.+ Check Woocommerce > Wc Cancel for cancellation requests you received.== Frequently Asked Questions === Where can I get support or talk to other users ? =If you get stuck, you can ask for help in
